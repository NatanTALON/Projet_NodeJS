Fonction appelée lors d'une requête http post sur la page d'authentification. Elle vérifie que l'utilisateur est bien dans la base de données et que ses identifiants sont corrects. postRandom(req, res)

Fonction appelée lors d'une requête http post sur la page du jeu Random. Cette fonction compare le score envoyé dans la requête avec le meilleur score de l'utilisateur. Si le record est battu, la fonction vérifie le classement de l'utilisateur avec les 5 meilleurs joueurs de la communauté et met à jour le tableau des meilleurs joueurs si besoin.
